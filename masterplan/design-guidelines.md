# design-guidelines.md

## Emotional thesis

Feels like a printed concert ledger taped to a concrete wall ---
neutral, documentary, anonymous.

## Typography

-   Neutral grotesk sans-serif
-   H1 (DATE): All caps, heavy
-   H2 (TITLE): All caps, bold
-   Body: 1.6--1.8 line height
-   Max width: 720--800px

## Color system

-   Background: #FFFFFF
-   Text: #000000
-   Divider: #EAEAEA No gradients. No shadows. WCAG AA+ contrast.

## Layout

-   8pt grid
-   Single centered column
-   Whitespace separation only
-   No cards or containers

## Motion

-   150--200ms subtle fades
-   Underline on hover
-   No scaling or bounce

## Voice

-   Neutral
-   Direct
-   Anonymous

Microcopy: - TODAY - TOMORROW - NO UPCOMING EVENTS - SOURCE
